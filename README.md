# Ruby Shader Project

A Unity project demonstrating a procedurally generated ruby gem shader with animated glow effects, Fresnel rim lighting, and gradient noise patterns.

## 🎮 How to Run the Project

1. **Download the Project**
   ```bash
   git clone <repository-url>
   ```
   *Or download as ZIP and extract to your desired location*

2. **Open in Unity**

3. **Play**
   - Click the **Play** button (▶️) in the Unity Editor toolbar
